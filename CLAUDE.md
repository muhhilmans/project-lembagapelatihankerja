# Alur Sistem 

- client membuat lowongan 
- mitra mencari lowongan dan ngeapply salah satu lowongan
- client menerima lamaran dari mitra sesuai jumlah yang dibutuhkan
- admin setting jadwal untuk interview yang sudah di terima oleh client
- status lamaran di mitra sudah menjadi penjadwalan
- lalu admin setting link interview dan deskripsi untuk mitra dan client
